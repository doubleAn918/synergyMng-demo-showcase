## 产品功能概览

### 1) 工作台
- 经营看板
- 库存预警（物料/产品）
- 待办审批

### 2) 组织架构
- 部门管理
- 岗位管理
- 员工管理
- 用户管理
- 角色管理（支持菜单权限分配）

### 3) 基础信息
- 单位管理
- 仓库管理
- 物料管理
- 产品管理
- BOM 管理（版本、默认 BOM、历史版本、改进）

### 4) 伙伴管理
- 客户管理
- 供应商管理

### 5) 采购与销售
- 采购单 / 采购退货单
- 销售单 / 销售退货单
- 提交、审核、驳回、查看、打印

### 6) 生产管理
- 生产单
- 完工确认
- 质检管理
- 生产成本记录

### 7) 库存管理
- 物料台账
- 产品台账
- 库存流水
- 库存盘点

### 8) 财务中心
- 应收对账台账 / 应收对账 / 收款单
- 应付对账台账 / 应付对账 / 付款单

### 9) 系统管理
- 菜单管理
- 编号规则
- 系统参数
